## Asset Guidelines

- **Resolution:** minimum 1280×720 for screenshots, 1920×1080 preferred.
- **Format:** PNG for static images, GIF or MP4-to-GIF for demos.
- **Compression:** optimize images before committing to keep the repository size reasonable.
- **Accessibility:** if images contain critical data, provide a brief caption or alt text in the README.
